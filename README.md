# Клиент для данных по Супергероям

### Возможности
- Поиск самого высокого супергероя по полу и наличию работы


## Инструкция по использованию
### Запуск приложения
Требуется Python версии 3.14.2+

Установка осуществляется через терминал:

git clone https://github.com/DobavNebo/heroes.git

cd heroes
pip install -r requirements.txt

### Запуск пользовательского интерфейса
1) Введите в терминале python UI.py
2) Следуйте инструкциям в терминале

### Запуск функции
1) Импортируйте функцию find_tallest из core.py в ваш проект любым удобным для вас способом
2) Вызовите функцию find_tallest
    - Входные данные (2 значения): пол(гендер)(строка), булевое значение(есть постоянная работа или нет)
    - Выходные данные (1 значение): ответ в виде словаря, содержащего данные о герое

### Тестирование
1) Введите в терминале python test.py
2) Следуйте инструкциям в терминале


P.S. Более сложные тесты чем представленные невозможно осуществить в виду того, что формулировка задания подразумевает конкретную сигнатуру функции, а для более точного выявления ошибок необходимо возвращать и значение ошибки, что требует иметь дополнительный вывод в сигнатуре.
